# webpy_demo
Securities data analysis demo by webpy
- data.py： 从 `tushare` 接口导入数据到 `MySQL` 数据库
- Code.py:：`web.py` 运行的主文件，其中功能类包含实现的算法
- templates：模板的目录

|编程环境||
| :--: | :--: |
| 语言 | Python |
| ide | PyCharm |
| 系统 | Windows10 |
| 数据接口 | [tushare](https://tushare.pro/) |
| 框架 | [web.py](https://webpy.org/docs/0.3/tutorial.zh-cn) |
